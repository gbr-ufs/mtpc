# Inteligência Artificial na Universidade Federal de Sergipe: Como o Seu Uso pode Comprometer o Aprendizado em Disciplinas Introdutórias de Programação

***Artigo para a disciplina "Métodos e Técnicas de Pesquisa para Computação"***

# Sumário

Artigo científico escrito para a obtenção dos créditos da disciplina "Métodos e Técnicas de Pesquisa para Computação". Disserta sobre como o uso de ferramentas de inteligência artificial podem comprometer o processo de aprendizado de programação, mediante a análise das opiniões dos discentes da [Universidade Federal de Sergipe](https://www.ufs.br/).
